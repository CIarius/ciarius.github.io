In this example the data comes from the Oracle XE sample database. The data is returned as XML by the DBMS_XMLGEN function. The XML is transformed to HTML by a XSL stylesheet. The HTML is styled by a CSS stylesheet. The end result can be seen here. Please note that the rendering is slower because it is done here on the client and the orders/contacts links don't work because GitHub Pages only serves static content. In a production environment I would transform the XML on the server in a Java servlet similar to this servlet and let the client handle the CSS and JavaScript.
